# VedicRecon v1.0.a

VedicRecon is a lightweight, modular reconnaissance tool designed to simplify and automate the most repetitive phases of information gathering. Built for pentesters, cybersecurity learners, and bug-bounty hunters, it focuses on clarity, structure, and meaningful output.

---

## üìå Introduction

Reconnaissance often becomes noisy, scattered, and inefficient due to multiple tools, different scan modes, and unorganized results. VedicRecon aims to bring everything into one clean workflow:

- Host discovery  
- Port and service enumeration  
- OS and version detection  
- Exploit reference gathering  
- Report generation  

It is especially useful for beginners who want an easy, guided structure, and for intermediate testers who want fast, scriptable recon.

---

## ‚ú® Features

### *Host Discovery*
- ARP-based network scanning  
- Interface auto-detection  
- Live host listing with MAC vendor details  

### *Scan Profiles*
- Fast, full, stealth, UDP, web, OS, and version detection profiles  
- Unified XML output for consistent processing  
- Clean parsing of ports, services, and banners  

### *Exploit Enumeration*
- Searchsploit integration for exploit references  
- Noise filtering to avoid irrelevant matches  
- Clear mapping between services and known exploits  

### *CLI Modes*
- Guided mode for beginners  
- Basic CLI mode for automation or quick use  

### *Reporting*
- Structured TXT and XML outputs  
- HTML reports for readable documentation  
- Clean organization of results for write-ups or portfolio use
